// BRIEF

A mobile-first platform running 13 active brands on shared infrastructure. Each brand needed its own identity while sharing the same component base — brand-kit delivery was a month-long bottleneck, blocking the team from actual product work.

I joined as Lead Product Designer to build the system, lead the team, and ship the back-office tooling that operators depended on daily.

● Shipping

// APPROACH

Built the system with an atomic architecture and design tokens — primitive → semantic → brand layers. Each brand became a configuration file on top of a shared component base, not a separate library to maintain.

  • Set sprint priorities, ran design reviews, mentored team on systems thinking and engineering handoff, set up onboarding for new designers.
  • Used Microsoft Clarity and internal metric sources for research — usability testing, interview frameworks, and metric reviews with product and analytics. Decisions argued from data.
  • Designed end-to-end UX for a multi-role back-office used by 50–100 operators — multi-state workflows, content tooling, role-based cabinets for ops, support, finance, and compliance.
  • Built internal AI tools on Claude for rapid prototyping — cut prototyping time on new flows by ~30% and made AI part of the team's daily workflow.
